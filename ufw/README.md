# `ufw` setup

## Use

- Read [`setup-ufw.sh`](setup-ufw.sh) first and make any required edits!

```bash
sudo ufw status

chmod +x setup-ufw.sh
sudo ./setup-ufw.sh
sudo ufw status
sudo ufw enable
```
